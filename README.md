# ITQAN: Decentralized Offline-First Educational Ecosystem

## Overview
ITQAN is an innovative, decentralized educational platform engineered to bridge the knowledge retention gap (the "Knowledge Evaporation" phenomenon) among high school students. Utilizing advanced Spaced Repetition algorithms (SM-2 variant), it ensures the transition of foundational knowledge into long-term memory, perfectly aligning with the Human Capability Development Program (Saudi Vision 2030).

## Architectural Positioning: Decentralized Offline-First
The ecosystem operates entirely on a decentralized client-side architecture. Both the Student Application and the Teacher Dashboard execute within the browser's isolated sandbox environment. 
- **Zero Operational Cost:** Eliminates the need for centralized cloud servers.
- **100% Availability:** Functions seamlessly without an active internet connection.

## Data Governance & Sovereignty: Privacy by Design
Built with strict adherence to national cybersecurity controls and data privacy standards.
- **Zero-Trust Storage:** All academic performance data, evaluations, and user identities are encrypted and stored locally via advanced `localStorage` mechanisms.
- **Psychological Security:** By preventing external data leaks or unauthorized access to academic records, the system safeguards the psychological well-being of struggling students.

## Automated Evidence Integrity
To ensure data integrity and prevent the forgery of academic evidence:
- **Client-Side Asset Processing:** The Teacher Dashboard automatically compresses images locally to optimize storage.
- **Automated Watermarking:** Applies an immutable "ITQAN" watermark to generated assets and reports before exportation, ensuring that all reports submitted to official arbitration committees are authentic and tamper-proof.
